# GardenLawn Finntheme

This is the Finntheme for the GardenLawn project.

## Overview

This theme is a child theme of the GardenLawn Hyvatheme, designed to provide specific styling and functionality for the Finn store view.

## Directory Structure

- **Magento_Theme**: Customizations for the Magento Theme module.
- **web**: Web assets (CSS, JS, images).
